# Property Pulse 🏠

Une plateforme web Full-stack de mise en relation entre particuliers pour la location de logements. Ce projet a été réalisé pour mettre en pratique les concepts avancés de Next.

## 🚀 Fonctionnalités

- **Authentification sécurisée** : Inscription et connexion des utilisateurs.

- **Gestion des annonces(Crud)** : Possibilité pour les utilisateurs connectés d'ajouter, modifier ou supprimer leurs habitations.

- **Moteur de recherche** : Recherche dynamique des logements disponibles

- **Messagerie interne** : Système de contact direct entre locataires et propriétaires.

## 🛠 Stack Technique

- **Frontend / Backend** : Next.js (App Router), React, JavaScript
- **Styling** : Tailwind CSS
- **Base de données** : MongoDB

## 💻 Installation et lancement

Pour lancer le projet en local , suivez ces étapes : 

1. Cloner le dépot : 

```bash
git clone https://github.com/Chino57/property-pulse.git
```

2. Installer les dépendances :
npm install

3. Configurer les variables d'environnement (créer un fichier .env.local et y ajouter vos clés, par exemple la chaine de connexion MongoDB).

4. Lancer le serveur de développement : 
npm run dev

5. Ouvrez http://localhost:3000 dans votre navigateur
